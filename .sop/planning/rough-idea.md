# Rough Idea

Build a clone of [jsoncrack.com](https://jsoncrack.com/) - a JSON visualization tool.

## Reference Features (from jsoncrack.com)

**Core Functionality:**
- Transform JSON into interactive graph visualizations
- Real-time visualization as user types
- Tree-view and graph-view formats

**Multi-Format Support:**
- JSON, YAML, CSV, XML, TOML file handling

**Format Conversion:**
- Convert between formats (JSON to YAML, XML to JSON, etc.)

**Code Generation:**
- TypeScript interfaces
- Golang structs
- Rust serde definitions
- JSON Schema generation

**Advanced Tools:**
- JWT decoding
- jq command execution
- JSON path operations
- Mock data generation from schemas

**Export:**
- PNG, JPEG, SVG export of visualizations

**Security:**
- All processing client-side (no server storage)
- ~300 KB data limit based on complexity
